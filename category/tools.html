Git常用操作

Posted on 2016-12-11 20:24 in Tools

对于开源项目的开发来说，熟练使用git的应该算是必备技能。本篇博客记录了平时工作中较常用的git操作。

git log

  • 显示所有历史记录
# git log
  • 显示最近N条历史记录
# git log -N
  • 显示详细历史记录，包括代码改动内容
# git log -p
  • 只显示一行，哈希值和提交说明
# git log --pretty=oneline
  • 显示提交历史记录的简单图形
# glog
# git log --pretty=format:"%H %s" --graph
  • 自定义格式显示历史记录
# git log --pretty=format:" "

常用的格式占位符写法及其代表的意义如下：

选项   说明
%H  提交对象（commit）的完整哈希字串
%h  提交对象的简短哈希字串
%T  树对象（tree）的完整哈希字串 …
